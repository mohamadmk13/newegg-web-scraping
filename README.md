# Newegg Product Web Scraping & Data Cleaning & Analysis

## Overview
This project focuses on collecting product data from the Newegg website using web scraping techniques, followed by data cleaning and preprocessing to produce a structured and analysis-ready dataset.

The final cleaned data is exported into Excel format, making it suitable for further data analysis, reporting, or business insights.

---

## Objectives
- Scrape product information from Newegg
- Store raw scraped data in a structured format
- Clean and preprocess the dataset
- Export clean data for analysis and visualization

---

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook
- Excel (XLSX)

---

## Data Processing Steps
1. Scraping product data such as:
   - Product name
   - Price
   - Hardware information
   - Availability
2. Handling missing values
3. Formatting numeric columns
4. Removing duplicates
5. Exporting clean data to Excel
